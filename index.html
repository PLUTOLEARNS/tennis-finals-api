Tennis Finals API

Welcome to the Tennis Finals API! This API provides Wimbledon Men's Singles final results.

Available Endpoints

Root

GET / - API information

Try it: /.netlify/functions/api

Wimbledon Results

GET /wimbledon?year=YYYY - Get Wimbledon final results by year

Parameters: year (required) - Year between 1877 and 2030

Try it: /wimbledon?year=2023

Or: /wimbledon?year=2022

Health Check

GET /health - Check API health status

Try it: /health

Example Response

{
  "year": 2023,
  "champion": "Novak Djokovic",
  "runner_up": "Carlos Alcaraz",
  "score": "1–6, 7–6(6), 6–1, 3–6, 6–4",
  "sets": 5,
  "tiebreak": true
}

Features

  • Real-time data from Wikipedia
  • Historical data from 1877 onwards
  • JSON response format
  • CORS enabled
  • Rate limiting and security
